# 🚀 intro_cdt: Jornada de Transformação Tecnológica

Bem-vindo ao repositório **intro_cdt**! Este é o ponto de partida da sua jornada no desenvolvimento de software. Aqui, você sairá do zero absoluto até a capacidade técnica de construir sistemas funcionais, organizados e persistentes.

Este bloco inicial combina a base de ferramentas essenciais para o mercado de trabalho (Git, GitHub e Metodologias Ágeis) com um mergulho profundo na linguagem Python.

---

## 📅 Estrutura do Repositório

O conteúdo está organizado em blocos evolutivos de aprendizagem:

### 🛠️ Bloco 1: Ambientação e Cultura Dev
* **Versionamento de Código:** Domínio de Git e GitHub para gerir e partilhar projetos de forma profissional.
* **Metodologias Ágeis:** Introdução aos frameworks **Scrum** e **Kanban**, aprendendo a organizar o fluxo de trabalho como nas grandes empresas de tecnologia.

### 🐍 Bloco 2: Fundamentos de Python & Lógica
* Sintaxe básica e tipos de dados.
* Lógica de programação com estruturas condicionais (`if`, `elif`, `else`) e loops de repetição.

### 🗃️ Bloco 3: Estruturas de Dados (Módulo 04)
Organização de dados complexos através das estruturas nativas do Python:
* **Listas `[]`:** Coleções ordenadas e **mutáveis**. Exemplo de aplicação no sistema de gerenciamento de frutas com validação de duplicados.
* **Tuplas `()`:** Coleções ordenadas e **imutáveis**. Utilizadas para dados fixos, como os dias da semana.
* **Dicionários `{}`:** Estruturas baseadas em **pares chave-valor** e **mutáveis**, ideais para mapeamento rápido (como o cardápio da semana).
* **Conjuntos/Sets `{}`:** Coleções de elementos únicos, excelentes para buscas rápidas.

### 💾 Bloco 4: Funções & Persistência de Dados
* Criação de funções para modularização e reaproveitamento de código.
* Manipulação de ficheiros locais (**CSV** e **JSON**), permitindo guardar e consultar informações de forma persistente.
